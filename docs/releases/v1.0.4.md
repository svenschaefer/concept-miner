# Release Notes: v1.0.4

Status: planned

## Scope

- Final strict-stream hard cut for legacy compatibility and fallback behavior.

## Changes

- Removed CLI compatibility commands:
  - `concept-miner run`
  - `concept-miner validate`
- Removed legacy mode aliases:
  - `generic_baseline`
  - `default_extended`
- Default-extended mode now hard-fails if wikipedia-title-index is unavailable.
- Step12 ingestion now requires `mentions[]` and does not fall back to token-only extraction.
- Runtime seed artifact loading now resolves strict mode-specific files only:
  - `seed.concept-candidates.13a.yaml`
  - `seed.concept-candidates.13b.yaml`

## Validation

- unit and integration tests updated for strict behavior.
- full `npm run ci:check` required before publish.
